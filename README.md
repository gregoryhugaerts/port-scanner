# port scanner

Becode port scanner python project

## Usage

### hatch

- install hatch

```sh
pip install hatch
```

- run with `hatch run scanner`
- run tests with `hatch run test:test`

### regular python

- install dependencies with `pip install .`
- run from src folder: `python -m port_scanner`
